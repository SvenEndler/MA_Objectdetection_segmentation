# MA_Objectdetection_segmentation
Dieses GitHub Repository enthält den Code und die Ergebnisse zur Masterarbeit mit dem Thema automatisierte Erkennung und Objektsegmentierung in Karten. Das System kombiniert etablierte Objekterkennungsverfahren mit einem Segmentierungsverfahren. Das Ziel ist es Symbole und Piktogramme möglichst genau zu zu segmentieren und auszugeben.
